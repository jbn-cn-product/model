# Car & Face Recognition System

## 功能特性

- **人脸检测**：基于 RetinaFace，支持高精度人脸定位与关键点检测。
- **人脸特征提取**：提取 512 维人脸特征向量，支持清晰度评分（拉普拉斯方差）。
- **车牌检测**：支持多角度车牌定位，输出 4 个角点坐标。
- **车牌识别**：支持透视变换矫正，可识别蓝牌、绿牌、黄牌等多种车牌及颜色。
- **鲁棒性**：支持中文路径、透明背景图片（RGBA）、批量处理。

## 模型详细信息

| 类名 | 模型文件 | 功能 | 分辨率 (宽x高) | 模型大小 | 内容要求 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **FaceDetector** | `face_det.onnx` | 人脸检测 | **640x640** | **1.94 MB** | 需要保持图像原始比例，使用 Letterbox 方式（缩放+填充灰色/黑色）处理，以确保检测框和关键点准确。 |
| **CarDetector** | `car_det.onnx` | 车牌检测 | **640x640** | **3.65 MB** | 同人脸检测，需保持原始比例进行 Letterbox 处理，确保能准确定位车牌的四个角点。 |
| **Feature** | `face_rec.onnx` | 人脸特征提取 | **112x112** | **166.3 MB** | **必须经过对齐矫正**。根据检测到的5个关键点进行仿射变换，只保留人脸核心区域，去除背景干扰，确保五官位置固定。 |
| **PlateRecognizer** | `car_rec.onnx` | 车牌识别 | **168x48** | **702 KB** | **必须经过透视变换**。根据检测到的4个角点将倾斜的车牌“拉正”，裁剪出纯车牌区域，不能包含车身背景。 |

## 环境依赖

- Python 3.8+
- OpenCV (`opencv-python`)
- NumPy
- ONNX Runtime (`onnxruntime` 或 `onnxruntime-gpu`)
- Pillow (可选，用于中文绘制)

## 快速开始

1.  **准备图片**：将待检测图片放入 `imgs` 文件夹。
2.  **运行程序**：
    ```bash
    python main.py
    ```
3.  **查看结果**：
    - 终端输出识别到的车牌号、颜色、人脸置信度及清晰度分数。
    - 处理后的图片（带检测框）保存在 `results` 文件夹中。
